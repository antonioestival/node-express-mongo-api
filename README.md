# Objetivo

Projeto exemplo para iniciar projetos de API, com NodeJS, Express, MongoDB e JWT

## Up & Runnig

Ambiente de Dev utilizando vagrant

### Pré Requisitos
Vagrant
Virtual Box

### Vagrant
vagrant up
vagrant ssh
cd /vagrant

### Build de Produção
yarn build
yarn start
